User needs

  • I need to produce an appeal response that is accurate and comprehensive so that my reasoning is clear and I avoid direction notices. 
    • I need to know what an appeal response must include, so that I can  produce a comprehensive response.
    • I need to refer to the correct up-to-date legislation to justify my decision so that my response is grounded in legislation.
    • I need to show which evidence I’ve used to make my decision, so that it’s clear how I’ve considered the evidence.
    • I need to address all the appellant’s grounds for appeal, so that it’s clear I’ve considered and responded to all their points. 
  • I need to produce an appeal response that’s appropriate for the people who will read it, so that they can understand why I’ve made the decision. 
    • I need to know who the appeal response is for, so that I can write my response in a way that’s suitable for them. 
    • I need to tailor my response so that it’s appropriate for the people who’ll read it and reflects the details of a particular case. 
    • I need to write my response in a clear and concise way that avoids duplication so that it does it's job and looks professional.

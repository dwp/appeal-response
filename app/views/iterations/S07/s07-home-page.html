Home page

Hypothesis / Experiment

  • We have seen that the appeal writing process can take a long time
  • We think this is because taking decisions requires them to go through the entire case history in different systems
  • So if we allow AWs to save their responses
  • We will see that the AWs can edit the same response over different sessions
  • We'll know this is true when the appeal writers will be able to pause and resume writing an appeal.

What's new or changed?

  • A home page to start the journey
  • A list of previously created appeal responses

What did we learn?

  • So far this seems to be a strong concept.
  • Neither participant raised any objections to the layout or content.
  • List-based presentation of work items was familiar to both Ps.
  • One P referenced having something similar currently – but which lets you do more.
  • One P mentioned inclusion of filters.
  • One P asked about access management – e.g. what if someone off sick and doing someone else’s cases?
  • “I could potentially see myself using this. It gives you a breakdown of where you need to go.”
